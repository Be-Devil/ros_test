rgb2xyz

RGB to XYZ color space

[x, y, z] = rgb2xyz(r, g, b) xyz = rgb2xyz(rgb)

convert (R,g,b) coordinates to (X,Y,Z) color space. If RGB (or R, g, b) have more than one row, then computation is

done row wise.
