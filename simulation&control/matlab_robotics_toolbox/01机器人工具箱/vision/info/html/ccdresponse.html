ccdresponse

CCD spectral response

R = ccdresponse(lambda) is the spectral response of a typical silicon imaging sensor at the wavelength lambda [m]. The response is normalized in the range 0 to 1. If lambda is a vector then R is a vector of the same length whose elements are the response at the corresponding element of lambda.

Notes

  • Deprecated, use loadspectrum(lambda, 'ccd') instead.

References
