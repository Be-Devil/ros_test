isamesize

Automatic image trimming

out = isamesize(im1, im2) is an image derived from im1 that has the same dimensions as im2. This is achieved by cropping and scaling.

out = isamesize(im1, im2, bias) as above but bias controls which part of the image is cropped. bias=0.5 is symmetric cropping, bias&lt;0.5 moves the crop window up or to the left, while bias&gt;0.5 moves the crop window down or to the right.
