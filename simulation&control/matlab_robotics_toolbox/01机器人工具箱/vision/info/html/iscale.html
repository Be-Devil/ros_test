iscale

Scale an image

out = iscale(im, s) is a version of im scaled in both directions by s which is a real scalar. s&gt;1 makes the image larger, s&lt;1 makes it smaller.

Options

'outsize', S set size of OUT to HxW where S=[W,H]
'smooth', S initially smooth image with Gaussian of standard deviation S (default 1). S=[] for no smoothing.

See also

ireplicate, idecimate, irotate
