ksobel

Sobel edge detector

k = ksobel() is the Sobel x-derivative kernel: 

|-1  0  1|
|-2  0  2|
|-1  0  1|

Notes

  • This kernel is an effective horizontal edge detector
  • The Sobel vertical derivative is K'
